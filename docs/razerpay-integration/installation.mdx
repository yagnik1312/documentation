# Installing Razorpay

Learn how to integrate **Razorpay** into your Application.

## Getting Started

To integrate Razorpay, you'll need a Razorpay account and the following details:

- `Key ID`
- `Key Secret`

You can get these from the Razorpay Dashboard after creating an account and setting up a new app.

### What you'll need

- Node.js version 18.0 or above
- Razorpay account credentials

## Backend Integration

### Step 1: Install Razorpay Node.js SDK

First, you need to install the Razorpay Node.js SDK. Navigate to your backend directory and run:

npm install razorpay
